# Campora - PHP Satış Sitesi

Campora, PHP ve MySQL kullanılarak geliştirilmiş basit bir e-ticaret / satış sitesidir.

## 🚀 Özellikler
- Ürün listeleme
- Sepet sistemi
- Kullanıcı giriş / kayıt
- Admin panel (varsa ekle)
- MySQL veritabanı bağlantısı

## 🛠️ Kullanılan Teknolojiler
- PHP
- MySQL
- HTML / CSS
- JavaScript
- XAMPP (yerel sunucu)
